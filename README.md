## ‚è¨ Installation

- Clone repo using:
```bash
git clone https://github.com/ivan-resetnikov/That-Level-Again
```

### No Python

- Launch using:
```bash
cd "That-Level-Again\bin\main.exe"
```

- Install required libraries using:
